# Matériel Laboratoire Disponible

## Microcontrôleurs et calcul

- Arduino (Uno, Mega, Nano)
- Raspberry Pi (3B+, 4)
- ESP32 / ESP8266 (IoT, WiFi intégré)
- Postes informatiques

## Capteurs

- Température / humidité : DHT22, BME280
- Luminosité : LDR, BH1750
- Humidité du sol : capteurs capacitifs
- Distance : ultrason HC-SR04
- Divers selon disponibilité

## Actionneurs

- Servomoteurs (SG90, MG996R)
- Moteurs DC + drivers (L298N, L293D)
- Relais (modules 1, 2, 4 canaux)
- Pompes miniatures

## Communication

- WiFi (intégré ESP32, dongles USB)
- Bluetooth (modules HC-05/06)
- LoRa (capacité de base)
- MQTT comme protocole IoT standard

## Énergie

- Panneaux solaires (5-50W)
- Batteries (Li-ion, plomb)
- Régulateurs de charge
- Outils de mesure de puissance
- Composants d'électronique de puissance de base

## Fabrication

- Imprimante 3D (PLA, PETG)
- Outillage à main et électroportatif
- Matériel de prototypage électronique (platines, fer à souder, composants passifs)
- Découpe laser (accès limité)

## Logiciels

- Modélisation SysML : Papyrus, Modelio ou équivalent
- Programmation : Arduino IDE, PlatformIO, Python
- CAO : SolidWorks, Fusion 360 ou FreeCAD
- Dashboard IoT : Node-RED (sur Raspberry Pi)
- Broker MQTT : Mosquitto
